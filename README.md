Quad
===

Syntax
---
`quad a b c`

Installation
---
    git clone https://github.com/DovgaNik/Quad.git
    cd Quad
    sudo make install

Uninstallation
---
    git clone https://github.com/DovgaNik/Quad.git
    cd Quad
    sudo make uninstall
Or you can type

    sudo rm /bin/quad
